# PIC16F877A-Embedded-C-Projects

This repository contains bare-metal Embedded C firmware and peripheral drivers developed for the **PIC16F877A** (8-bit Microcontroller). All drivers are written using **MPLAB XC8** compiler and validated via **Proteus VSM** simulation.

---
## 🛠️ Key Skills & Technologies Demonstrated
* **Programming Language:** Embedded C
* **Microcontroller Architecture:** Microchip PIC16F877A (8-bit)
* **IDE & Compiler:** MPLAB X IDE / MPLAB v8, XC8 Compiler
* **Simulation Tool:** Proteus VSM
* **Peripherals Interfaced:** 
  * GPIO Configuration (LED, Push Buttons, LCD)
  * Analog-to-Digital Converter (ADC)
  * Universal Asynchronous Receiver-Transmitter (UART)
  * Pulse Width Modulation (PWM) & Timers

---
## Directory Overview
* `01_GPIO_Blink/` - LED blinking and push-button polling routines.
* `02_LCD_Display/` - 16x2 LCD display driver (4-bit / 8-bit mode).
* `03_ADC_Read/` - Multi-channel ADC conversion for sensor interfacing.
* `04_UART_Comm/` - Serial communication driver for PC-MCU data transfer.
* `05_PWM_Control/` - Hardware PWM generation for DC motor speed control.
